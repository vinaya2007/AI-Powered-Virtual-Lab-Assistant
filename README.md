🧠 AI-Powered Virtual Lab Assistant

An interactive web-based assistant designed to help engineering students perform lab experiments efficiently with AI guidance, structured content, and downloadable resources.

## 🚀 Features

* 💬 **AI Chatbot**

  * Ask doubts related to experiments
  * Get responses like Aim, Theory, Procedure, Output
  * Step-by-step guidance during lab

* 🎤 **Voice Input**

  * Speak directly to the chatbot
  * Hands-free interaction

* 📚 **Experiment Support**

  * Covers multiple ECE lab experiments
  * Smart keyword-based experiment detection

* 📄 **Downloadable Records**

  * Each experiment has its own PDF
  * Hosted using Google Drive links

* 🖼️ **Circuit Diagrams & Graphs**

  * Visual understanding of experiments
  * Displayed directly inside chatbot

* 🎯 **Viva Preparation**

  * AI-generated viva questions
  * Helps in lab exams

* 📱 **Responsive UI**

  * Works on desktop and mobile
  * Sidebar with hamburger menu

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **AI Integration:** Google Gemini API
* **Storage:** JSON + Google Drive

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/vinaya2007/AI-Powered-Virtual-Lab-Assistant.git
```

2. Navigate to project folder:

```
cd lab-assistant
```

3. Install dependencies:

```
pip install flask python-dotenv requests
```

4. Add your Gemini API key

5. Run the app:

```
python app.py
```

6. Open in browser:

## 🔑 Important Notes

* Images use Google Drive **view links**
* PDFs use Google Drive **download links**
* Make sure all files are:
  👉 **"Anyone with the link" enabled**
